School Classes Data upload
Read the following instruction to ensure you smooth use of the data upload feature on this page.

  • the file must not be larger than 4mb in size
  • the structure of data import must follow a strict set described in the downloadable file below.
  • if school does not give students a unique registration number, the column for registration number could be left blank. However it should not be deleted from the worksheet.
